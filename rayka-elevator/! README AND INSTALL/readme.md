# Rayka elevator creator

A simple and optimized elevator script for FiveM servers.

---

## 🚀 How to Install (Easy Setup)

**Step 1: Import SQL**
* Import `rayka_elevators.sql` into your database.

**Step 2: Configure**
* Open `config.lua` and set your **Framework** and **Notification** system.

**Step 3: Start Resource**
* Add `ensure rayka_elevators` to your server.cfg and restart your server.
* Enjoy the script!
